						<h2> Массивы </h2>
						<p> Массивы - это группа однотипных элементов, имеющих общее имя и расположенных в памяти рядом. Также они ассоциируются с матрицей, таблицей  </p>

						<li> Все элементы имеют один тип </li>
						<li> Весь массив имеет одно имя </li>
						<li> Все элементы в памяти расположены рядом</li>
					</td>
				</tr>
				<tr>
					<td>
						<h3> Одномерный массив </h3>
						<p> Одномерный массив - это конечная последовательность данных одного типа, имеющих общее имя. Каждый отдельный компонент массива называется элементом. Количество элементов называется размером массива. Тип элементов определяет тип массива. Размер и тип массива указываются при его описании в разделе var или type. Ниже приведено описание массива 'A', элементы которого являются типом 'Real' </p>

						<textarea id = "code-frame" readonly>
//Вариант 1
VAR
a:array [1..1000] of Real;

// Вариант 2
Const N = 1000 // Описали константу (переменную, значение которой нельзя менять в ходе программы)
VAR a:array [1..N] of Real;	</textarea>
					</td>
				</tr>

					<td>
						<p> Номер элемента называется индексом. Индексы могут быть целыми положительными константами или целыми переменными.	<br>
							Чтобы обратиться к некоторому элементу массива, нужно рядом с идентификатором массива (именем) в квадратных скобках указать индекс элемента.
						</p>
						<p>
							Например, A[99] - это 99-ый элемент массива 'A'
						</p>
